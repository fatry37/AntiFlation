Configuration requise

⚙️ Connecte ta base de données Supabase
1
Va sur supabase.com → crée un compte gratuit et un nouveau projet
2
Projet → Settings → API → copie l'URL et la clé anon public
3
Dans SQL Editor, exécute :

create table prix (
  id bigint generated always as identity primary key,
  produit text not null,
  prix numeric not null,
  enseigne text not null,
  quartier text,
  created_at timestamptz default now()
);
alter table prix enable row level security;
create policy "lecture publique" on prix for select using (true);
create policy "ajout public" on prix for insert with check (true);
4
Ouvre index.html dans un éditeur texte et remplace METS_TON_URL_ICI et METS_TA_CLE_ICI
